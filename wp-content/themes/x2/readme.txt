Create any type of blog, shop, magazine, social network or portfolio site.

== Installation ==


= Automatic Install from the WordPress Theme Repository =

Install like any other WordPress theme from the WordPress theme repository.

1. Go to your dashboard to Appearance Themes and search the Theme Repository for "x2".

2. Install the theme and click on "activate".

That's it!


= Manual Installation =

1. Go to your admin dashboard to Appearance -> Themes

2. Click on "upload"

3. Upload the zipped theme file from your computer

4. When uploaded and installed, click on "activate"..

And you're finished :)


= Updating =

Enjoy the automatic updates a you know by the WordPress Theme Repository!

Make a backup before to be sure you will not affect your live production site.
